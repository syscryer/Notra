# 00 产品定义

## 用户价值

Notra 服务的是长期在 Windows 上处理脚本、配置、日志、SQL、代码片段的用户。核心场景不是写大型工程，而是快速打开、查找、批量替换、检查差异和保存。

## 核心功能优先级

P0：

- 秒开应用。
- 打开、保存、另存为。
- 多 Tab。
- 当前文件查找和替换。
- 打开文档查找和替换。
- 查找结果高亮和结果列表。
- 目录查找和替换预览。

P1：

- 最近文件。
- 会话恢复：最近文件、打开文档、工作目录、搜索历史和主题。
- 暗色主题。
- 搜索历史。
- 替换历史。

P2：

- 简单语法强调。
- 大文件只读模式。
- 搜索收藏。

不做：

- 插件。
- 宏。
- FTP。

## Notepad++ 体验基准

- 查找窗口的选项直接可见，不把常用选项藏进设置。
- 搜索模式支持普通、扩展和正则。
- 目录搜索支持文件类型过滤和跳过目录。
- 结果面板按文件和行显示，点击能回到对应文档。
- 替换动作需要明确告诉用户替换了多少处。
- 关闭未保存文档前必须给保存、不保存、取消选择。

## v0.1 完成标准

- 用户可以完成一个真实工作流：打开目录，搜索关键字，查看结果，批量预览替换，确认写回文件。
- 当前文件查找替换使用 Notepad++ 风格浮动窗口，操作连贯。
- UI 信息密度足够，主窗口默认干净，编辑区始终是主角。
