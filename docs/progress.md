# 进度记录

## 当前状态

- 项目方向已确定：漂亮、便捷、速度快。
- 核心卖点已确定：搜索与替换体验。
- 风格参考已确定：Catio 外壳、mxterm 密度、Notepad++ 搜索替换。
- Trellis 生命周期入口已迁入 Notra 仓库。
- Rust workspace 已落地：`notra-core` 提供文档、编码、搜索替换、目录扫描能力，`notra-app` 已切换为 Tauri + Monaco 桌面 UI。
- 当前文件搜索、替换当前、全部替换、目录搜索、目录替换预览、确认写回已经可用。
- 搜索/替换输入已改为 Notepad++ 风格浮动工具窗，不再常驻主界面底部。
- 主窗口已调整为 Notepad++ 式干净结构：无常驻左侧文件树、无右侧工具轨，底部结果面板默认隐藏并按需显示。
- 主编辑区已改为铺满剩余空间，去除默认输入框黑色边框。
- 打开文档范围搜索和打开文档全部替换已经可用，结果按 tab 分组。
- 关闭未保存文档时会提示保存、不保存、取消。
- 关闭应用时会汇总所有未保存文档，并提供全部保存并退出、不保存退出、取消。
- 支持保存全部，快捷键 `Ctrl+Shift+S`。
- 支持命令行传入文件或目录，启动后直接打开或设为工作区。
- 打开目录会作为目录搜索上下文保存，但不在主窗口展开文件树。
- 文件系统只读或超过 20MB 编辑阈值的文档会进入只读保护，替换和保存入口会提示跳过原因。
- 会话持久化已可用：恢复打开文件、最近文件、工作目录、搜索历史、替换历史、主题、底部面板、Markdown 预览和搜索选项。
- 最近文件菜单已可用，可从工具栏快速重新打开并清空历史。
- 支持另存为，`Ctrl+Shift+S` 会触发当前文件另存。
- 支持打开文档范围查找和打开文档范围替换。
- 关闭未保存 Tab 和关闭窗口时提供保存、不保存、取消三种选择。
- 目录树支持展开/折叠，并会随会话保存折叠状态。
- 命令行启动参数已接入：传入文件会直接打开，传入目录会设置为工作区。
- 查找结果会高亮命中的文本片段。
- 只读文档在 Monaco 中会进入真实只读模式，保存或替换会给日志提示。
- 保存快照已接入，撤销回保存内容时会自动清除 Tab 脏标记。
- 查找窗口内展示查找历史、替换历史和搜索收藏，点击即可回填。
- 搜索收藏已可用，会随会话恢复。
- 搜索模式支持普通、扩展和正则；扩展模式支持 `\r`、`\n`、`\t` 等转义。
- 目录树支持刷新，目录查找支持递归、隐藏项、文件过滤和跳过目录配置。
- 目录搜索支持递归子目录开关、隐藏文件开关、文件过滤、跳过目录和最大文件大小限制。
- 目录搜索会跳过疑似二进制文件，目录替换预览会跳过只读文件。
- 保存已有文件和目录替换写回已经使用临时文件、备份和失败回滚流程。
- 目录搜索、目录替换预览和目录替换写回改为后台任务，避免 UI 假死。
- 结果列表点击会打开文件并定位命中，当前命中会在编辑区高亮和选中。
- 扩展模式替换支持 `\r`、`\n`、`\t` 等转义，正则替换支持 `$1` 捕获组。
- 性能烟测通过：12 万命中内存查找约 458ms，内存替换约 452ms，160 文件目录搜索约 110ms。

## 下一步

继续压测大文件、大目录和 UI 细节，补齐更多 Notepad++ 风格的边界行为，并加强查找窗口的键盘操作细节。
