# 03 编辑核心

## 已实现能力

- 文档打开、保存、另存为。
- 保存全部。
- 命令行启动可以直接打开文件或目录。
- 命令行传入目录后会设为目录搜索上下文。
- UTF-8、UTF-8 BOM、UTF-16LE、UTF-16BE、GBK 兼容解码。
- 换行符识别：LF、CRLF、CR。
- 脏状态。
- 保存快照：撤销回保存内容时自动清除脏状态。
- Undo/Redo 快照。
- 行列统计。
- 文件大小统计。
- 已有文件保存使用临时文件写入和替换流程，降低保存中断时写坏文件的风险。
- 文件系统只读或超过 20MB 编辑阈值的文件会进入只读保护。
- 只读文档允许查找和复制，不允许保存、编辑、替换、撤销和重做写入。
- 关闭应用时会检查所有未保存文档，并提供全部保存并退出、不保存退出、取消。

## 验收点

- 打开文本后状态栏显示编码、换行符、行列数和大小。
- 修改后 Tab 出现脏标记。
- 保存后脏标记消失。
- 保存全部会保存所有 dirty 文档。
- 通过命令行传入文件时，应用启动后会直接打开该文件。
- 通过命令行传入目录时，应用启动后会直接设置为工作区。
- 只读文档不会参与保存全部，界面会提示跳过原因。
- 替换全部可以撤销。
- 撤销回保存内容后脏标记消失。
- 关闭未保存 Tab 前会提供保存、不保存、取消。
- 关闭应用前会保护所有未保存文档。
